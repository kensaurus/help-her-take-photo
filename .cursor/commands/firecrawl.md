# Firecrawl Research

## Overview
Use Firecrawl scraping service to research online documentation, implementation methods, latest trends, and troubleshooting information for current development tasks.

## Steps
1. **Define research scope**
   - Identify specific technology or problem to research
   - Determine key search terms and topics
   - Set research objectives and success criteria

2. **Execute Firecrawl searches**
   - Search official documentation sites
   - Scrape Stack Overflow for common issues
   - Research GitHub repositories for examples

3. **Analyze implementation patterns**
   - Review best practices and recommended approaches
   - Compare different implementation methods
   - Collect code examples and snippets

## Firecrawl Research Checklist
- [ ] Research objectives defined clearly
- [ ] Official documentation reviewed
- [ ] Community discussions analyzed
- [ ] Code examples collected
- [ ] Best practices identified
